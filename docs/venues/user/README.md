# DojoPool User Guide

## Welcome to DojoPool

Welcome to DojoPool, the next generation of pool gaming! This guide will help you get started with our innovative system that combines traditional pool with cutting-edge technology.

## Getting Started

### 1. Creating Your Account

1. **Download the App**
   - iOS: App Store
   - Android: Google Play
   - Web: app.dojopool.com

2. **Sign Up**
   - Enter your email
   - Create a password
   - Verify your account
   - Complete your profile

3. **Profile Setup**
   - Add profile picture
   - Set skill level
   - Choose preferred games
   - Connect social accounts

### 2. Finding Venues

1. **Venue Discovery**
   - Use venue map
   - Filter by location
   - Check availability
   - View amenities

2. **Venue Details**
   - Operating hours
   - Table availability

   - Special events

3. **Making Reservations**
   - Select date and time
   - Choose table
   - Add players
   - Confirm booking
